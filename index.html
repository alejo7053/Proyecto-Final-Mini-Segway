Introducción

Actualmente muchos sistemas cuentan con un control realizado por medio de alguna tarjeta programable o microprocesador, este proyecto es la implementación de un mini segway. Un segway es un vehiculo de transporte ligero giroscópico eléctrico de dos ruedas, con autobalanceo controlado por ordenador. El Segway es el primer dispositivo de transporte con autobalanceo. El ordenador y los motores situados en la base mantienen la base del Segway horizontal todo el tiempo. El usuario se debe inclinar hacia la dirección que quiera tomar (delante, atrás, derecha o izquierda). El motor es eléctrico y silencioso. Entonces en este proyecto se realiza un modelo a escala de este vehiculo, utilizando un Arduino Leonardo el cual posee un microcontrolador Atmega32U4, un giroscopio (ref. MPU6050), un puente H (ref. L293D) y un motor DC de 9V con 9500rpm.
